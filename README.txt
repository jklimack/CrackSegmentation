

DIRECTORIES
---------------------------------------------------------

DeepCrack
Provided train and test data. 

output
Predicted results from the implemented network. 


SCRIPTS
---------------------------------------------------------

CrackSegmentation.m
Main script file containing the network, used for training the network. Must be ran before 
either of the other scripts, and the variables cannot be cleared in between scripts. 

create_output.m
This script is used to save the predicted labelled images to files in the output directory. 

AVPR_test.m
This script is used for obtaining the statistical results of the network. 


OTHER FILES
---------------------------------------------------------

AVPR_Klimack.pdf
Report. 

cracknet_final.mat
Final trained network that was used for results.
